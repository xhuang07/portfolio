Dark & Sweet

Project Type
Book Design
Tools
Indesign
Photoshop
Convert a “low-brow” theme TV show and its contents into an beautifully bound book. This project tests the limits of how a person interacts with a book to digest the contents.
This book analyze each character from two angles, one is about the hidden “dark” side, the other shows the “sweet” side. I adopted the french fold technique meticulously to ‘hide’ one book inside another, on it’s hardcover there is a pill-shape button which can be turned either vertically or horizontally. Depends on which direction user chose unconsciously, they opened up a book which presents a unique perspective about this show.
